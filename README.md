## How to run (dev)
1. Clone the repo: 
```bash
git clone https://github.com/spuckhafte/twitter-clone
```
2. Open two terminals and goto the app directory in both of 'em.
```bash
cd twitter-clone
```
3. First terminal:
```bash
cd frontend
npm run dev
```
4. Second Terminal:
```bash
cd backend
npm test
```
5. Open `http://localhost:5173/` in your browser.
